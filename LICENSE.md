# Licenza e uso dei dati

AtlasPI segue un modello open core.

Principi:
- il core del progetto sarà pubblicato con una licenza open source
- eventuali componenti premium devono restare separati dal core
- i dataset importati mantengono le rispettive licenze originarie
- ogni fonte utilizzata deve essere tracciabile e documentata
- nessun dato deve essere incluso nel progetto senza verifica della licenza e attribuzione corretta

Nota:
la licenza definitiva del core deve essere definita prima del primo rilascio pubblico.
La compatibilità tra licenze dei dataset geografici e distribuzione pubblica dell'API
deve essere verificata esplicitamente prima della pubblicazione.
