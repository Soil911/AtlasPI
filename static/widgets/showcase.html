AtlasPI Embeddable Widgets

Embed historical data on any website. Copy the code, paste it into your HTML.

Entity Card

Display a card for any historical entity with name, dates, capital, and confidence score.

<iframe src="https://atlaspi.cra-srl.com/widget/entity/1" width="400" height="300"></iframe>
ParameterDescription
/entity/{id}Entity ID (integer)
?theme=lightSwitch to light theme

Mini Timeline

Display a chronological list of events within a date range.

<iframe src="https://atlaspi.cra-srl.com/widget/timeline?year_min=-500&year_max=500" width="800" height="300"></iframe>
ParameterDescription
year_minStart year (negative for BCE). Default: -3000
year_maxEnd year. Default: 2025
limitMax events. Default: 50
themelight for light theme

On This Day

Show historical events that occurred on today's date (or a specific date).

<iframe src="https://atlaspi.cra-srl.com/widget/on-this-day" width="400" height="400"></iframe>
ParameterDescription
dateSpecific date as MM-DD (e.g., 07-14). Default: today
themelight for light theme
